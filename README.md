A **Prison Management System** built to efficiently manage prisoner records, staff details, and administrative operations. This project aims to digitize and simplify prison management using modern web technologies.

---

## 🚀 Features

- 👮‍♂️ Admin dashboard for managing prison data
- 🧑‍⚖️ Add, update, and delete prisoner records
- 📋 Manage staff details
- 🔍 Search and filter prisoners
- 🔐 Secure authentication system
- 📊 Organized data handling with database integration

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- HTML5, CSS3
- JavaScript (ES6+)

### Backend:
- Spring Boot
- REST APIs

### Database:
- MySQL
